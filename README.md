# Ohmyfood!

![Aperçu de l'application](assets/images/ohmyfood.png)
- Ce projet a été réalisé dans le cadre du projet n°4 de la formation Intégrateur Web d’OpenClassrooms.

- OhMyFood est une startup qui ambitionne de devenir un acteur majeur du secteur de la gastronomie.
- L’objectif du site est de permettre aux utilisateurs de :
	- • consulter les menus de restaurants gastronomiques,
	- • réserver une table en ligne,
	- • sélectionner leurs plats à l’avance afin de réduire leur temps d’attente sur place.

## Outils et languages pour la réalisation du projet

- Le projet à été réalisé en **HTML5/CSS3** ainsi que le framework front end **SASS**
